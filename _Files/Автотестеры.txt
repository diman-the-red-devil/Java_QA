Требуемый опыт работы: не требуется

Компания «Консист Бизнес Групп» запускает конкурс на стажировку с последующим приемом на постоянную работу –
это твой шанс попасть в уникальную команду профессионалов, которые реализуют полный комплекс ИТ-сервисов, включая разработку и внедрение информационных систем, а также услуги в области бизнес-консалтинга на территории России, стран СНГ и Европы.

Эта программа для тебя, если
•	Ты студент 4-го курса бакалавриата, магистратуры, или уже выпускник технических вузов по специальностям, близким к прикладной информатике\математике
•	Ты стремишься работать в ИТ
•	Возможно, у тебя есть желание сменить профессию/переквалифицироваться
•	Готов обучаться не менее 15-20 часов в неделю в ближайшей перспективе
•	У тебя аналитический склад ума, тебе нравится разбираться в процессах, предлагая решения

Что тебя ждет?

•	Комплексная программа по подготовке тестировщиков автоматизаторов, включая теоретические и практические знания
•	Опытные преподаватели и наставники, которые будут определять твой план развития
•	Гибкий график (3 раза в неделю по неск. часов) для тех, кто учится или работает
•	По окончанию стажировки лучших оформляем в штат компании на позицию младшего тестировщика ПО.
•	После прохождения курса сможете автоматизировать тестирование и вывести карьеру в QA на новый уровень.

Чему тебя научим?

•	Работать с фреймворком JUnit
•	Писать UI автотесты с применением Selenium WebDriver 
•	Писать API автотесты с применением RestAssured 
•	Писать BDD автотесты с применением Cucumber 

Что от тебя ожидаем?

Обязательные (минимальные) навыки:

1. Знание основ языка программирования Java (либо C#)
- базовые конструкции
- объектно-ориентированное программирование
- обобщённое программирование
- коллекции
- исключения

2. Знание основ теории тестирования
- жизненный цикл продукта и тестирование
- типы, виды, уровни тестирования
- принципы, этапы тестирования
- техники тест дизайна
- тестовая документация

Дополнительные (будут плюсом) навыки:

3. Знание основ Git
- репозитории
- коммиты
- ветвление и слияние

4. Знание основ Maven
- зависимости, репозитории
- жизненный цикл и фазы
- плагины

5. Разное
- опыт работы с баг трекерами (напр Jira)
- знание SOAP и REST (владение инструментами SoapUI, Postman)
- знание SQL (умение строить запросы)
