# Цель

[![Содержание](https://img.shields.io/badge/-Содержание-1177ff)](#содержание)

* Познакомиться с подходами в разработке ПО
* Узнать что такое **TDD** и **BDD**
* Познакомиться с подходами в разработке тестов
* Узнать что такое **DDT** и **KDT**
* Посмотреть какие **BDD** фреймворки существуют
* Изучить основы **BDD** фреймворка **Cucumber**
* Научиться создавать проект с **JUnit 5**, **Selenium WebDriver 4** и **Cucumber**

***

# Результаты

* Понимание различных подходов в разработке ПО
* Понимание различных подходов в разработке тестов
* Умение создавать проект с **JUnit 5**, **Selenium WebDriver 4** и **Cucumber**
